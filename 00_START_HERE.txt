🎓 FRAMEWORK OVERVIEW:

KIMVIWARE implements a 4-phase pipeline for automated test generation:

Phase 0: Validation & Detection
  • Language detection
  • Structure validation

Phase 1: Trajectory Extraction
  • AST-based path extraction
  • Control flow graph generation
  • Symbolic constraint collection

Phase 2: SGATS Path Reduction
  • State-based trajectory clustering
  • Similarity analysis (Jaccard distance)
  • Coverage preservation (≥95%)
  • Results in 80-90% path reduction

Phase 3: EvoPath-GA Optimization
  • Multi-objective genetic algorithm
  • Population-based optimization
  • Results in additional 33-50% reduction

Phase 4: Execution & Mutation Testing
  • Concrete test generation (Z3 solver)
  • Mutation testing (6 operators)
  • Quality assessment (>90% target)
